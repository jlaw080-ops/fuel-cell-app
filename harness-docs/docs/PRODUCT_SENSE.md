# PRODUCT_SENSE.md — 제품 감각

## 이 앱의 사용자는 누구인가?
- 건축설비 엔지니어 (비개발자)
- 연료전지 설치 경제성을 평가하는 업무를 수행
- 엑셀로 하던 작업을 자동화하고 싶어한다

## 사용자가 가장 중요하게 생각하는 것
1. **계산 정확성**: 수식이 맞아야 한다 (잘못된 결과는 실무에서 큰 문제)
2. **빠른 비교**: 조건을 바꿔가며 여러 시나리오를 비교하고 싶다
3. **리포트 품질**: 클라이언트에게 제출할 수 있는 수준의 문서

## 사용 시나리오
1. 새 프로젝트 생성
2. 연료전지 모델 선택, 수량 입력
3. 운전시간 조건 설정
4. 계산 결과 확인 (에너지, 수익)
5. 경제성 지표 확인 (NPV, IRR, 회수기간)
6. AI 검토 의견 참고
7. PDF 리포트 다운로드
