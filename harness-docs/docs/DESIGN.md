# DESIGN.md — UI/UX 디자인 규칙

## 디자인 원칙

1. **일관성**: 색상, 폰트, 간격이 전체 앱에서 통일
2. **독창성**: 기본 템플릿 느낌이 아닌, 의도적인 디자인 선택
3. **기능성**: 사용자가 추측 없이 작업을 완료할 수 있어야 함
4. **전문성**: 엔지니어링 보고서에 어울리는 깔끔한 톤

## 색상 팔레트

> 구체적인 색상은 Phase 2에서 확정 (ui-ux-pro-max 스킬 적용)

- Primary: 에너지/신재생 테마 (그린/블루 계열)
- Secondary: 보조색
- Neutral: 배경, 텍스트
- Accent: 강조, 경고

## 타이포그래피

- 제목: 굵은 산세리프
- 본문: 가독성 높은 산세리프
- 숫자/데이터: 고정폭(monospace) 폰트 사용 권장

## 레이아웃

- 탭 네비게이션: 상단 고정
- 콘텐츠 영역: 최대 폭 1200px, 중앙 정렬
- 테이블: 줄무늬 배경, 숫자 우측 정렬
- 입력 폼: 라벨 상단 배치, 그룹별 카드 구분

## 리포트 디자인 (PDF)

- A4 세로형 (210mm × 297mm)
- 여백: 상하좌우 20mm
- 목차 포함
- 페이지 번호 하단 중앙
- 표: 경계선 회색, 헤더 배경색 적용
