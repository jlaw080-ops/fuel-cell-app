# 기술 부채 추적기 (Tech Debt Tracker)

> 실패한 작업, 미해결 문제, 개선 필요 사항을 기록한다.
> 가비지 컬렉션 시 이 파일을 확인하고 처리한다.

## 미해결 항목

| 날짜 | 분류 | 설명 | 상태 |
|------|------|------|------|
| — | — | (아직 기록 없음) | — |

## 분류 기준
- `버그`: 잘못된 동작
- `린트`: 린트 규칙 추가 필요
- `수식`: 수식 구현 오류 (최우선 처리)
- `문서`: 문서-코드 불일치
- `성능`: 속도/효율 개선 필요
- `UI`: 디자인 불일치
