# 계산 수식 정의서

> ⚠️ 이 파일의 수식은 절대 변경 금지 (CLADE.md 최우선 금지 규칙)
> 코드 구현 시 이 수식을 그대로 따른다.

---

## 용어 정의

- `sets[]`: 연료전지 세트 배열. 각 세트는 하나의 모델+수량 조합
- `정격발전용량_kW`: 라이브러리에서 호출된 값
- `열생산용량_kW`: 라이브러리에서 호출된 값
- `가스소비량_kW`: 라이브러리에서 호출된 값
- `설치수량`: 사용자 입력
- `중간부하운전시간`: 기본값 18시간, 사용자 수정 가능
- `최대부하운전시간`: 기본값 6시간, 사용자 수정 가능
- `해당월일수`: 월별가동일라이브러리에서 호출

---

## 입력값 계산

### 총설치용량
```
총설치용량_kW = Σ (sets[i].정격발전용량_kW × sets[i].설치수량)
```

---

## (1) 전력 생산량

```
일일_중간부하_전력생산량_kWh = Σ(sets[i].정격발전용량_kW × sets[i].설치수량) × 일일_중간부하_운전시간
일일_최대부하_전력생산량_kWh = Σ(sets[i].정격발전용량_kW × sets[i].설치수량) × 일일_최대부하_운전시간
월간_중간부하_전력생산량_kWh = 일일_중간부하_전력생산량_kWh × 해당월일수
월간_최대부하_전력생산량_kWh = 일일_최대부하_전력생산량_kWh × 해당월일수
```

## (2) 열 생산량

```
일일_열생산량_kWh = Σ(sets[i].열생산용량_kW × sets[i].설치수량) × (중간부하운전시간 + 최대부하운전시간)
월간_열생산량_kWh = 일일_열생산량_kWh × 해당월일수
```

## (3) 도시가스 사용량

```
일일_도시가스사용량_kWh = Σ(sets[i].가스소비량_kW × sets[i].설치수량) × (중간부하운전시간 + 최대부하운전시간)
월간_도시가스사용량_kWh = 일일_도시가스사용량_kWh × 해당월일수
```

---

## (4) 발전 수익

```
전기_기본요금_대비수익 = Σ(sets[i].정격발전용량_kW × sets[i].설치수량) × 기본요금
중간부하_발전수익 = 월간_중간부하_전력생산량_kWh × 중간부하_전기요금
최대부하_발전수익 = 월간_최대부하_전력생산량_kWh × 최대부하_전기요금
발전_월간총수익 = (전기_기본요금_대비수익 + 중간부하_발전수익 + 최대부하_발전수익) × 부가세(1.1) × 전력기금(1.032)
발전_연간총수익 = Σ(1월~12월 발전_월간총수익)
```

> 전기요금: `전기요금라이브러리.json`에서 해당월의 중간부하, 최대부하 값 호출

## (5) 열 생산 수익

```
월간_열생산수익 = 월간_열생산량_kWh × 도시가스요금_일반용
열생산_연간총수익 = Σ(1월~12월 월간_열생산수익)
```

> 도시가스요금: `가스요금라이브러리.json`의 "일반용(영업1) 도시가스 요금" 값

## (6) 도시가스 사용 요금

```
월간_도시가스사용요금 = 월간_도시가스사용량_kWh × 연료전지전용요금
연간_도시가스사용요금 = Σ(1월~12월 월간_도시가스사용요금)
```

> 연료전지전용요금: `가스요금라이브러리.json`의 "연료전지전용" 값

## (7) 최종 수익

```
월간_수익 = 발전_월간총수익 + 월간_열생산수익 - 월간_도시가스사용요금
에너지생산_최종수익 = Σ(1월~12월 월간_수익)
```

---

## 경제성 평가 조건

```
총수익 = 에너지생산_최종수익
초기투자비용 = Σ(sets[i].정격발전용량_kW × sets[i].설치수량 × sets[i].kW당설치단가)
연간유지보수비용 = Σ(sets[i].정격발전용량_kW × sets[i].설치수량 × sets[i].kW당연간유지비용)
```

## 연도별 경제성 (output3)

```
연도별_순현금흐름[0] = -초기투자비용
연도별_순현금흐름[n] = 총수익 - 연간유지보수비용  (n >= 1)
누적순현금흐름[n] = Σ(연도별_순현금흐름[0..n])
```

## 최종 평가 (output4)

- **단순투자회수기간**: 누적순현금흐름이 0 이상이 되는 최초 연도
- **NPV (순현재가치)**: 할인율 적용한 현금흐름의 현재가치 합
- **IRR (내부수익률)**: NPV = 0이 되는 할인율
- **B/C ratio**: 총 편익 현재가치 / 총 비용 현재가치

> 할인율, 분석기간 등 세부 파라미터는 사용자 입력 또는 기본값으로 처리
