# 제품 요구사항 (Product Specs)

## 문서 목록

| 파일 | 설명 |
|------|------|
| `calculation-formulas.md` | 계산 수식 정의서 (⚠️ 수정 금지) |

## 앱 핵심 요구사항

1. 연료전지 정보 입력 (라이브러리에서 선택 + 수량 직접 입력)
2. 운전시간 정보 입력 (유형 선택 + 시간 직접 입력)
3. 에너지 생산량/사용량 자동 계산 (월별)
4. 에너지 생산 수익 자동 계산 (월별/연간)
5. 경제성 평가 (NPV, IRR, 투자회수기간)
6. AI 검토 의견 (Gemini API, A4 1페이지 이내)
7. 리포트 생성 (화면 보기 + PDF 다운로드, A4 세로형)

## 라이브러리 JSON 파일 목록

- `연료전지제품라이브러리.json`: 형식 → 제조사 → 모델명 → 제품정보
- `월별가동일라이브러리.json`: 연간운전유형 → 월별 가동일수
- `전기요금라이브러리.json`: 월별 중간부하/최대부하 전기요금
- `가스요금라이브러리.json`: 일반용(영업1) 요금, 연료전지전용 요금
